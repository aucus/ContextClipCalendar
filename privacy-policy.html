ContextClipCalendar 개인정보 보호 정책

1. 개인정보처리방침 개요

ContextClipCalendar("당사", "저희", "우리")는 사용자의 개인정보 보호를 최우선으로 생각합니다. 이 개인정보처리방침은 당사가 수집, 사용, 보호하는 개인정보에 대해 설명합니다.

최종 업데이트: 2024년 12월

적용 범위: 이 개인정보처리방침은 ContextClipCalendar Chrome 확장 프로그램 및 관련 서비스에 적용됩니다.

2. 수집하는 개인정보

2.1 자동 수집 정보

  • 기본 사용 정보: 확장 프로그램 사용 통계, 오류 로그, 성능 데이터
  • 기술적 정보: 브라우저 버전, 운영체제 정보, 확장 프로그램 버전, 디바이스 정보
  • 서비스 이용 정보: 기능 사용 빈도, 오류 발생 시점, 성능 지표

2.2 사용자 제공 정보

  • Google 계정 정보: Google Calendar 연동 시 필요한 기본 계정 정보 (이메일 주소, 사용자 ID)
  • 설정 정보: 사용자가 설정한 개인화 옵션, 기본 캘린더 선택
  • 권한 정보: Google Calendar API 접근 권한

2.3 수집하지 않는 정보

  • 사용자가 선택한 웹페이지 텍스트 내용
  • 개인 캘린더 이벤트 내용
  • 웹사이트 방문 기록
  • 개인 식별 가능한 민감한 정보
중요: 당사는 사용자가 선택한 웹페이지 텍스트 내용을 수집하지 않습니다. 모든 클립 데이터는 Google Calendar API를 통해 사용자의 Google Calendar에만 직접 저장되며, 당사 서버에는 전송되지 않습니다.

3. 개인정보 사용 목적

3.1 주요 사용 목적

  • 서비스 제공: ContextClipCalendar 기능 제공 및 개선
  • Google Calendar 연동: 사용자가 선택한 텍스트를 Google Calendar에 저장
  • 기술적 지원: 오류 해결 및 성능 최적화
  • 사용자 경험 개선: 서비스 품질 향상을 위한 분석
  • 보안: 부정 사용 방지 및 보안 강화

3.2 세부 사용 목적

  • 인증 및 권한 관리: Google 계정을 통한 사용자 인증 및 Calendar API 접근 권한 관리
  • 서비스 개선: 사용 패턴 분석을 통한 기능 개선 및 새로운 기능 개발
  • 고객 지원: 사용자 문의 및 기술적 문제 해결
  • 법적 준수: 관련 법령 및 규정 준수

4. 개인정보 공유 및 제공

4.1 공유하지 않는 원칙

당사는 다음과 같은 경우를 제외하고는 사용자의 개인정보를 제3자와 공유하지 않습니다:

  • 사용자의 명시적 동의가 있는 경우
  • 법적 요구사항에 따른 경우
  • Google Calendar API를 통한 정상적인 서비스 제공

4.2 Google 서비스와의 연동

  • Google Calendar API: 사용자가 선택한 텍스트는 Google Calendar API를 통해 직접 사용자의 Google Calendar에 저장됩니다.
  • Google OAuth: 사용자 인증을 위해 Google OAuth 2.0을 사용합니다.
  • 데이터 접근 제한: 당사는 사용자의 Google Calendar 데이터에 직접 접근할 수 없습니다.

4.3 제3자 서비스

  • Chrome 웹 스토어: 확장 프로그램 배포 및 업데이트
  • GitHub: 소스코드 호스팅 및 이슈 관리
  • GitHub Pages: 웹사이트 호스팅
중요: 사용자가 선택한 텍스트는 Google Calendar API를 통해 직접 사용자의 Google Calendar에 저장되며, 당사는 이 데이터에 접근할 수 없습니다. 모든 데이터 전송은 Google의 보안 정책에 따라 암호화됩니다.

8. 쿠키 및 추적 기술

8.1 사용하는 기술

당사는 다음과 같은 기술을 사용합니다:

  • 로컬 스토리지: 사용자 설정 저장
  • 세션 스토리지: 임시 데이터 저장

8.2 사용하지 않는 기술

당사는 사용자 추적을 위한 쿠키나 기타 추적 기술을 사용하지 않습니다.

8.3 제3자 추적

Google Analytics나 기타 추적 도구를 사용하지 않으며, 사용자의 웹사이트 방문을 추적하지 않습니다.
