프로그램 소개

ContextClipCalendar란?

ContextClipCalendar는 AI 기반 스마트 일정 등록 도구입니다. 복잡한 텍스트를 복사하고 클릭 한 번으로 Google Calendar에 일정을 자동 등록할 수 있습니다.

주요 특징

  • 간편한 사용법: 텍스트 복사 → 클릭 한 번으로 끝
  • AI 자동 분석: Gemini AI가 핵심 정보를 자동으로 추출
  • Google Calendar 연동: 자동으로 캘린더에 일정 등록
  • 다양한 소스 지원: 이메일, Slack, 카카오톡 등 어디서든 사용 가능

왜 ContextClipCalendar를 사용해야 할까요?

  • 시간 절약: 복잡한 일정 입력 과정을 단순화
  • 정확성: AI가 핵심 정보를 정확히 추출
  • 편의성: 어디서든 텍스트만 복사하면 바로 등록
  • 무료: Gemini API 무료 할당량으로 충분히 사용 가능

필요한 준비물

기본 요구사항

  • Google 계정 (Gmail 계정이면 됩니다)
  • Chrome 브라우저 (최신 버전 권장)
  • 인터넷 연결
  • 5-10분의 설정 시간

Gemini API 키 발급 방법

ContextClipCalendar가 AI 기능을 사용하기 위해 필요한 API 키를 발급받아야 합니다.

  1. Google AI Studio 접속

    Chrome 브라우저를 열고 다음 링크로 이동하세요:

    Google AI Studio 접속
  2. API 키 생성
    1. Google 계정으로 로그인
    2. "Create API key" 또는 "API 키 만들기" 버튼 클릭
    3. 프로젝트 선택 또는 생성:
      • 기존 프로젝트가 있다면: 드롭다운에서 선택
      • 프로젝트가 없다면: "Create in new project" 선택
  3. API 키 복사 및 저장
    1. 생성된 API 키를 클릭하여 전체 선택
    2. Ctrl+C (Windows) 또는 Cmd+C (Mac)로 복사
    3. 메모장이나 텍스트 파일에 붙여넣기하여 안전하게 저장
    4. 나중에 찾기 쉽게 "Gemini API Key"라고 제목을 붙이세요

중요한 보안 규칙

  • API 키는 비밀입니다: 다른 사람과 절대 공유하지 마세요
  • 안전한 저장: 공용 컴퓨터에 저장하지 마세요
  • 의심스러운 활동 발견 시: 즉시 Google AI Studio에서 API 키를 삭제하고 새로 생성하세요

API 키란?

API 키는 Google의 AI 서비스(Gemini)를 사용할 수 있는 디지털 열쇠입니다. ContextClipCalendar가 AI를 통해 텍스트를 분석하고 일정을 추출할 때 필요한 인증 정보입니다.

설치 방법

Chrome 웹 스토어에서 설치

ContextClipCalendar를 Chrome 웹 스토어에서 설치하세요.

Chrome 웹 스토어에서 설치
Chrome 웹 스토어에서 'Chrome에 추가' 클릭

확장 프로그램을 툴바에 고정

설치 후 주소창 오른쪽의 퍼즐 아이콘(확장 프로그램)을 클릭한 뒤, ContextClipCalendar 우측의 핀 아이콘을 클릭해 고정하세요.

퍼즐 아이콘 → ContextClipCalendar → 핀 아이콘 클릭

API 키 설정

ContextClipCalendar 아이콘을 클릭하여 팝업을 열고, 설정 페이지로 이동하여 API 키를 입력하세요.

팝업 인터페이스에서 설정 접근
옵션 페이지에서 Gemini API 키 입력
  1. 팝업에서 '설정' 버튼 클릭 또는 툴바의 ContextClipCalendar 아이콘을 우클릭 → '옵션' 클릭
  2. Gemini API 키 입력: 발급받은 API 키를 입력하고 '저장' 클릭
  3. API 키 테스트: 'API 키 테스트' 버튼으로 연결 상태 확인
  4. Google Calendar 연동: OAuth 인증을 통해 Google Calendar와 연동

설정 완료 체크리스트

  • ✅ ContextClipCalendar 확장 프로그램 설치 완료
  • ✅ 툴바에 확장 프로그램 고정 완료
  • ✅ Gemini API 키 입력 및 테스트 완료
  • ✅ Google Calendar 연동 완료

사용 방법

기본 사용법 (3단계)

  1. 텍스트 복사: 일정이 포함된 텍스트를 선택하고 복사 (Ctrl+C 또는 Cmd+C)
  2. 확장 프로그램 실행: 툴바의 ContextClipCalendar 아이콘 클릭
  3. 일정 등록: AI가 분석한 결과를 확인하고 Google Calendar에 등록

실제 사용 예시

예시 1: 이메일에서 일정 추출

원본 텍스트: "안녕하세요. 내일 오후 3시에 회의실 A에서 팀 미팅이 있습니다. 참석 부탁드립니다."

AI 분석 결과:

  • 제목: 팀 미팅
  • 시간: 내일 오후 3시
  • 장소: 회의실 A

예시 2: Slack 메시지에서 일정 추출

원본 텍스트: "다음주 화요일 2시에 클라이언트 미팅이 있어요. 강남역 근처 카페에서 만나요."

AI 분석 결과:

  • 제목: 클라이언트 미팅
  • 시간: 다음주 화요일 2시
  • 장소: 강남역 근처 카페

무료 버전으로 충분한 이유

1,000번의 일일 요청은 일반적인 개인 사용자에게 매우 충분합니다:

  • 하루에 10번의 일정 등록 = 100일 사용 가능
  • 하루에 50번의 일정 등록 = 20일 사용 가능
  • 대부분의 사용자는 하루에 5-20번 정도만 사용
  • 월간으로는 최대 30,000회까지 사용 가능 (일일 1,000회 × 30일)
