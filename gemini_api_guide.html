🧩 확장 프로그램 설치

아직 ContextClipCalendar를 설치하지 않았다면 아래 버튼으로 Chrome 웹 스토어에서 추가하세요.

Chrome 웹 스토어에서 설치
  1. 'Chrome에 추가' 클릭 → '확장 프로그램 추가' 확인
  2. 추가 후 주소창 오른쪽 퍼즐 아이콘에서 고정하면 더 편리해요
  3. 설치가 끝났다면 아래 '초간단 시작'으로 넘어가세요

📌 고정 및 옵션 열기 (스크린샷)

1

확장 프로그램을 툴바에 고정

주소창 오른쪽의 퍼즐 아이콘(확장 프로그램)을 클릭한 뒤, ContextClipCalendar 우측의 핀 아이콘을 클릭해 고정하세요.

2

아이콘 우클릭(또는 메뉴) → 옵션
3

옵션에서 Gemini API 키 입력

'Gemini API 키' 입력란에 복사한 키를 붙여넣고 '저장' → '테스트' 버튼으로 연결을 확인하세요.

💡 API 키란?

API 키는 Google의 AI 서비스(Gemini)를 사용할 수 있는 디지털 열쇠입니다. ContextClipCalendar가 AI를 통해 텍스트를 분석하고 일정을 추출할 때 필요한 인증 정보입니다.

🚀 단계별 API 키 발급

1

Google AI Studio 접속

Chrome 브라우저를 열고 다음 링크로 이동하세요:

🌐 Google AI Studio 접속

Google 계정으로 로그인하세요.

2

API 키 생성

  1. 화면에서 "Get API key" 버튼을 찾아 클릭하세요
  2. "Create API key" 또는 "API 키 만들기" 버튼을 클릭하세요
  3. 프로젝트 선택 또는 생성:
    • 기존 프로젝트가 있다면: 드롭다운에서 선택
    • 프로젝트가 없다면: "Create in new project" 선택
3

API 키 복사 및 저장

  1. 생성된 API 키를 클릭하여 전체 선택하세요
  2. Ctrl+C (Windows) 또는 Cmd+C (Mac)로 복사하세요
  3. 메모장이나 텍스트 파일에 붙여넣기하여 안전하게 저장하세요
  4. 나중에 찾기 쉽게 "Gemini API Key"라고 제목을 붙이세요

🎯 일반인 사용자에게 충분한 이유

1,000번의 일일 요청은 일반적인 개인 사용자에게 매우 충분합니다:

  • 하루에 10번의 일정 등록 = 100일 사용 가능
  • 하루에 50번의 일정 등록 = 20일 사용 가능
  • 대부분의 사용자는 하루에 5-20번 정도만 사용

🔧 문제 해결

자주 발생하는 문제들

❌ "API 키가 유효하지 않습니다" 오류

해결 방법:
  • API 키를 다시 복사해보세요
  • 앞뒤 공백이 없는지 확인하세요
  • API 키가 완전히 복사되었는지 확인하세요

❌ "권한이 없습니다" 오류

해결 방법:
  • Google 계정으로 제대로 로그인했는지 확인
  • API 키 생성 시 올바른 프로젝트를 선택했는지 확인
  • 잠시 후 다시 시도해보세요

❌ "할당량을 초과했습니다" 오류

해결 방법:
  • 무료 사용 한도를 초과한 경우입니다
  • 24시간 후 다시 시도하거나 유료 플랜을 고려하세요
  • Google AI Studio에서 사용량을 확인해보세요

❌ "API 키를 찾을 수 없습니다" 오류

해결 방법:
  • Google AI Studio에서 API 키가 정말 생성되었는지 확인
  • API 키를 다시 복사해서 입력해보세요
  • 브라우저를 새로고침하고 다시 시도해보세요

🔄 여전히 문제가 있다면?

  1. 브라우저 새로고침: F5 키를 눌러 페이지를 새로고침하세요
  2. 확장 프로그램 재시작: Chrome에서 확장 프로그램을 비활성화 후 다시 활성화하세요
  3. 다른 브라우저 시도: Edge나 Firefox에서도 테스트해보세요
