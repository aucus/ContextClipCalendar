📅 ContextClipCalendar 설정

AI 기반 스마트 일정 등록을 위한 설정을 관리하세요

🤖 AI API 설정

Google AI Studio에서 발급받은 Gemini API 키를 입력하세요. 이 키는 일정 정보 추출과 AI 분석에 사용됩니다.

💡 Gemini API 키 발급 방법

  1. Google AI Studio에 접속하세요
  2. Google 계정으로 로그인하세요
  3. "Get API key" 버튼을 클릭하세요
  4. 새 API 키를 생성하거나 기존 키를 선택하세요
  5. 생성된 API 키를 위 입력란에 붙여넣으세요

🗓️ Google Calendar 연동

Google Calendar API를 사용하려면 먼저 OAuth 클라이언트를 설정해야 합니다.
OAuth 설정 확인 중...

🔑 Google Calendar OAuth 설정 방법

  1. Google Cloud Console에서 새 프로젝트를 생성하세요
  2. "API 및 서비스" → "라이브러리"에서 Google Calendar API를 활성화하세요
  3. "API 및 서비스" → "사용자 인증 정보"에서 "사용자 인증 정보 만들기" → "OAuth 클라이언트 ID"를 선택하세요
  4. 애플리케이션 유형을 "Chrome 확장 프로그램"으로 선택하세요
  5. 확장 프로그램 ID를 입력하세요 (chrome://extensions/ 에서 확인 가능)
  6. 생성된 클라이언트 ID를 복사하여 manifest.json 파일에 추가하세요: 
    "oauth2": {
  "client_id": "YOUR_CLIENT_ID.apps.googleusercontent.com",
  "scopes": [
    "https://www.googleapis.com/auth/calendar",
    "https://www.googleapis.com/auth/calendar.events"
  ]
}
  7. 확장 프로그램을 다시 로드하세요

📖 사용 가이드

🚀 ContextClipCalendar 사용법

  1. 텍스트 복사: 일정으로 등록할 텍스트를 클립보드에 복사하세요
  2. 확장 프로그램 실행: 브라우저 툴바에서 ContextClipCalendar 아이콘을 클릭하세요
  3. 자동 로드: 클립보드의 텍스트가 자동으로 로드됩니다
  4. 일정 등록: "일정 등록" 버튼을 클릭하여 Google Calendar에 등록하세요
  5. 컨텍스트 메뉴: 웹페이지에서 텍스트를 선택하고 우클릭으로도 이용 가능합니다

✨ AI가 자동으로 처리하는 내용

  1. 일정 제목 추출: 텍스트에서 적절한 일정 제목을 자동 생성
  2. 날짜/시간 분석: "내일 오후 3시", "다음주 월요일" 등 자연어 시간 표현 인식
  3. 장소 정보: 회의실, 주소, 온라인 플랫폼 등 장소 정보 추출
  4. 참석자 정보: 이메일 주소 기반 참석자 목록 생성
  5. 내용 요약: 긴 텍스트를 일정 설명에 적합하게 요약
